# Terminal Tetris

Play tetris in the terminal!

## Supported platforms

Linux. Any POSIX compliant OS might also work.

## Dependencies

* gcc v13.x
* ncurses

## Build

#### Debug 
`make`

#### Optimized
`make prod`

## Run
`./tetris`

## Controls

* Start: S
* Movement: arrow keys
* Rotate left: A
* Rotate right: D
* Quit: Q
